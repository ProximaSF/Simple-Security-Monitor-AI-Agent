# For AWS EC2 Ubuntu

## Objective:

The goal of this "activty" is to understand how AWS CloudWatch function and how to set it up. The end goal of this is to create a AI agent for security purpose (read system logs, file, etc). 
I have provided my process of setting up the CloudWatch. I have also attached another markdown file for the console agent configuration simplpe setup to connect AWS using the free-tier instance service. 
Note: The instance image I used is Ubuntu. Thus, the process of installing AWS CLI or the CloudWatch agent is a bit different: requires downloading a package first (.deb). Additionally, instead of `yum`, it's `apt` to install packages.
